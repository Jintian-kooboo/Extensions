The custom tool kit for kooboo cms


The extensions for kooboo CMS. The extension will be include:

1. Custom controls.
2. Custom content manager service.
3. Custom page plugins.
4. Some other extension.
5. Some Modules for Kooboo CMS.

Easy to use: Just need to copy the dlls into the bin folder of kooboo CMS.

How to use Cascading Control:
1. Add a setting name "Parent" indicate using which parent field in the CustomSetting.
2. Add a setting name "Folder" indicate using which folder as data source in the CustomSetting.
3. The DefaultValue as "Default message to select an option. if you set an empty value then does not shows any initial text.".
